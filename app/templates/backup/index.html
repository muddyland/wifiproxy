Create Backup

Downloads a .json file containing your WiFi network profiles, DHCP & bridge settings, DHCP reservations, Tailscale configuration, and WireGuard tunnel configs.

The backup file contains plaintext credentials (WiFi passwords, Tailscale auth keys). Store it securely and delete it when no longer needed.
Restore from Backup

Upload a backup file to restore WiFi networks, DHCP settings, reservations, Tailscale configuration, and WireGuard tunnels. Existing data will be replaced. Your admin password is not affected.

This will overwrite all current WiFi networks, DHCP reservations, and VPN configurations immediately.
What's included in a backup
WiFi Networks
SSIDs, passwords, priorities, auto-connect settings
DHCP / Bridge
Gateway, DNS, IP range, lease time, static reservations
WireGuard
Tunnel configuration files
Tailscale
Login server, auth key, route settings

Not included: admin password, active DHCP leases, iptables rules (regenerated from DHCP config), NetworkManager connection state.

Confirm Restore

This will overwrite all current WiFi networks, DHCP reservations, and VPN configurations with the contents of the uploaded backup.

This action cannot be undone.
